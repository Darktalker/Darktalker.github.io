C与C++

C —> 面向过程

C++ —> 面向对象，将C中的data/function封装成data member/function member

C++关于数据和函数

  • complex
    • 数据
      • 实部
      • 虚部
    • 函数
      • 加减乘除
      • 共轭，正弦
  • string
    • 数据
      • 字符
    • 函数
      • 拷贝，输出，附加，插入

C++ programs代码基本形式

  • .h (header files)

Chapter 03 构造函数

class的声明

inline内联函数

函数若在class body内定义完成，便自动成为inline候选人。内联类似于宏，执行速度可能会较快。

access level(访问级别)

  • public
  • private
  • protected

构造函数

不带指针的类多半不用写析构函数。

构造函数可以有很多个重载

一般函数也可以重载，void real(double r) const { re = r; }

Chapter 04 参数传递与返回值

把构造函数放在private区域内

把构造函数放在private区域内，表示该构造函数不能被外界调用，即不允许被外界创建对象，例如设计模式中的Singleton单例模式。
